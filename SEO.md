# Apex Prestige SEO configuration

Canonical origin: `https://www.apexprestige.ca`

## Indexable routes

- `/`
- `/soumission-services-exterieurs`
- `/confidentialite`
- `/nettoyage-exterieur-bromont`
- `/nettoyage-exterieur-magog`
- `/nettoyage-exterieur-orford`
- `/nettoyage-exterieur-rock-forest`

- `/nettoyage-exterieur-north-hatley`

- `/nettoyage-exterieur-canton-de-hatley`
- `/nettoyage-exterieur-saint-denis-de-brompton`

The sitemap is generated by `vite-plugin-sitemap`. The localized route data,
metadata, and indexability list are centralized under `src/consts/`.

Unknown `/nettoyage-exterieur-*` URLs receive an HTTP 404 through Netlify and the
React 404 view applies `noindex,follow`.
